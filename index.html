Conheça mais sobre mim

Sou graduando em Sistemas de Informação no IFES (Campus Serra) e atuo como Pesquisador em Formação (Bolsista FAPES), onde realizo pesquisa sobre Neuroevolução Gramatical aplicada a Redes Neurais e utilizei métodos estatísticos (ARIMA/SARIMA) para predição de dados - pesquisa que já resultou na publicação de dois artigos científicos.

Essa imersão acadêmica me proporcionou uma visão ampla. Para além da graduação, atuei em projetos envolvendo Desenvolvimento Front-End e Lógica de Programação em diversas linguagens. Somado a isso, a colaboração constante com colegas e professores fortaleceu meu trabalho em equipe e minha capacidade de resolver problemas de maneira proativa.

Toda essa trajetória, no entanto, começou muito antes. Minha jornada na tecnologia teve início aos 12 anos, quando encontrei no computador um refúgio para problemas pessoais. O que era apenas uma válvula de escape transformou-se em uma paixão duradoura pela computação. Inicialmente autodidata, explorei o universo do hardware, aprendendo sobre componentes e montagem — uma base que cultivo até hoje, enriquecida pelas lições de pessoas excelentes que cruzaram meu caminho.

Sou movido pela exploração constante. Tenho facilidade em navegar por tecnologias variadas — seja por exigência de projetos ou pelo simples prazer de "brincar" com algo novo e descobrir como funciona. Busco oportunidades para expandir meus conhecimentos em IA e desenvolvimento, acreditando na tecnologia como um meio de gerar impacto real.

NeuroGE

  • Python
  • TensorFlow
  • Neuroevolution
  • Pesquisa

Desenvolvimento da SANDL, uma linguagem declarativa hierárquica criada para descrever Redes Neurais. O projeto utiliza Neuroevolução Gramatical para automatizar o design de arquiteturas focadas em previsão de séries temporais multivariadas.

A solução propõe a eliminação da necessidade de configuração manual e superou modelos estatísticos tradicionais em testes de eficiência e precisão.

Classificação de Imagens com EuroSAT

  • Python
  • Computer Vision
  • CNN
  • Transfer Learning

Desenvolvimento de um sistema de Visão Computacional para classificar automaticamente o uso do solo utilizando imagens de satélite (Dataset EuroSAT).

Apliquei técnicas de Transfer Learning em redes CNN modernas (ResNet-50) para identificar 10 categorias de terreno, incluindo áreas urbanas e florestais. O modelo alcançou ~96-98% Acurácia e 0.1751 de loss, demonstrando alta eficiência para automação de monitoramento ambiental.

Laboratório de Séries Temporais

  • Python
  • Análise Estatística
  • ARIMA/SARIMA
  • AutoML

Um ambiente de experimentação focado na análise comparativa entre métodos estatísticos clássicos (ARIMA, SARIMA) e abordagens modernas de AutoML para previsão de dados.

O projeto documenta a implementação matemática dos modelos, testes de estacionaridade e avaliação de métricas de erro, servindo como base fundamental para pesquisas mais avançadas em Redes Neurais.

X-Ray Simulator

  • C Language
  • Estrutura de Dados
  • Linux
  • Memory Management

Desenvolvimento de um sistema de simulação de alto desempenho para gestão de fluxo de exames de Raio-X. O projeto implementa Tipos Abstratos de Dados (TADs) complexos, utilizando filas dinâmicas (FIFO) para orquestrar a alocação de pacientes em recursos limitados (máquinas) e lógica de prioridade para laudos médicos. O sistema inclui persistência de dados em arquivos e demonstra rigoroso controle de alocação de memória para evitar memory leaks.

Polinômio de Taylor

  • Python
  • Cálculo Numérico
  • Matplotlib
  • Math

Desenvolvimento de algoritmos para aproximação de funções matemáticas utilizando o Método de Taylor. O notebook explora a construção lógica da expansão em séries de potências, permitindo calcular valores de funções transcendentes com precisão ajustável.

O código inclui análises de erro e comparação de desempenho entre a implementação manual e as funções nativas das bibliotecas padrão.

Biblioteca de Árvores Balanceadas (AVL)

  • Java
  • POO
  • Complexidade de Algoritmos
  • Generics

Implementação de uma biblioteca genérica de Estruturas de Dados em Java, focada em Árvores Binárias de Busca (BST) e Árvores AVL (Auto-balanceáveis).

O projeto inclui a lógica de rotações automáticas para garantir complexidade de busca otimizada (O(log n)), superando problemas de degeneração de performance. A solução foi validada através de testes comparativos com grandes volumes de dados.
